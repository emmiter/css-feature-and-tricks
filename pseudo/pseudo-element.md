[CSS Pseudo-elements](http://www.w3schools.com/css/css_pseudo_elements.asp)

CSS伪元素用于样式化元素的特定部分

例如:

* 样式化元素的第一个字母，某一行
* 在元素内容的前面，后面插入插入内容

语法:

```css
selector::pseudo-element {
  property:value;
}
```

CSS3 所有的伪元素:

1. ::first-line
2. ::first-letter
3. ::before
4. ::after
5. ::selection [例子](http://www.w3schools.com/css/tryit.asp?filename=trycss3_selection)
